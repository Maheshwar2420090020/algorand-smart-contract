# Algorand Smart Contract Project

## 📌 Project Description
This project is developed as part of my coursework to explore Algorand blockchain and smart contract development. It focuses on building decentralized applications using Algorand.

## 🚀 Objectives
- Learn Algorand blockchain fundamentals
- Develop smart contracts using TEAL / PyTeal
- Build and test decentralized applications (dApps)

## 🛠️ Technologies Used
- Algorand Blockchain
- Python (PyTeal)
- GitHub for version control

## 📂 Project Structure
- `contracts/` → Smart contract code
- `scripts/` → Deployment scripts
- `README.md` → Project documentation
